The ultimate countdown block for Gutenberg, that help adding countdown functionality to your content easily.


== Description ==
This plugin is a feature rich countdown block made for Gutenberg.

Gutenberg is now a core feature so please update to WordPress v5, or install the <a href="https://wordpress.org/plugins/gutenberg/">Gutenberg plugin</a>.


= Plugin Features =

Works as any other Gutenberg block with the following settings:
- Square & Round styles
- Progress indicator
- Flexible digits && labels coloring and sizing options
- Flexible timer background coloring options
- Flexible border coloring options
- Animations


== Installation ==

Standard process for installing plugins. Only make sure you have the Gutenberg plugin installed.


== Development ==

Development location: https://github.com/bomsn/countdown-pro-for-gutenberg


== Frequently Asked Questions ==

= Can I remove one of the counters =

Yes, yes, there are options to enable or disable specific timer (days, hours, minutes, seconds).
